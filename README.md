# CO-SNE
code for CO-SNE: Dimensionality Reduction and Visualization for Hyperbolic Data (CVPR 2022)

# Dependencies

* numpy
* pillow
* scikit-learn==1.5
* scipy
* seaborn
* torch
* torchvision
* pip install geoopt

Please use pip to install all the packages.

# Running

```bash
$  python main.py
```
